What is recursion?

  1. Recursion is a function when they call itself until the reaches end point or it's a programming technique which a function call itself during the execution.
  2. They have two component, first base case and second is recursive case.
  3. In the base case : It's used stop the recursive and
  4. In the recursive case : It's used call itself. They passed the version of the problem as an argunment.
  5. Basically recursion is used for solving the complex problem by breaking them into the small sub-problem.
